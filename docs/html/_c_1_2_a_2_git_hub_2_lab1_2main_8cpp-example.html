Entry point of the graph application.

Entry point of the graph application.The program tries to load a graph from file. If the file does not exist, it offers the user to create a new graph (list or matrix). Afterwards, the user can repeatedly:

  • print the graph,
  • add or remove edges,
  • check connectivity,
  • compute distance between two vertices,
  • or recreate the graph.
Returns
Exit status code (0 for normal termination, 1 when exiting via menu).
Exceptions
None.

// Typical execution flow: // 1. User starts program. // 2. Chooses "Create new Graph as an Adjacency List". // 3. Adds edges and checks connectivity.

#include <iostream>
#include "graph.h"

cin.ignore();
grph.createGraphMatrix();

case '3':
return 1;
default:
cout << "Invalid Input...\n";

cin >> choice;
switch (choice)
{
case '1':
cin.ignore();
grph.createGraphList();
break;

cin.ignore();
grph.createGraphMatrix();
break;
case '3':
if (grph.getMatrix())
