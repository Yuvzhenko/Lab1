◆ checkVertices()

bool TestableGraph::checkVertices ( int & u,
int & v )
inline

Simplified vertex input validator for tests.

Reads a pair (u v) from standard input, validates that indices are within [0, vertices) or equal to the sentinel "-1 -1". If input is invalid (failbit set or out of range), prints an error message and returns false.

Unlike the production version, this function does not recurse: it only checks one pair of values.

Parameters
uReference where the first vertex index will be stored.
vReference where the second vertex index will be stored.
Returns
true if vertices are valid, false otherwise (including "-1 -1").
Exceptions
None.
